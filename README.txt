# AJ Chat Setup — OpenRouter (100% FREE, No Credit Card)

## Step 1: Get FREE API Key from OpenRouter
1. Go to: https://openrouter.ai
2. Click "Sign Up" (Google account use பண்ணலாம்)
3. Left menu → "API Keys" → "Create Key"
4. Copy the key (starts with sk-or-...)

## Step 2: Add key to key.txt
Open key.txt, delete everything, paste your key:
  sk-or-v1-xxxxxxxxxxxxxxxx

## Step 3: Run server
Open Terminal in this ajchat folder:
  node server.js

## Step 4: Open chat
  http://localhost:3000
